# 阿里百炼API Key申请简洁步骤（文档版）

## 一、前置准备

1. 拥有阿里云账号，完成实名认证（个人/企业均可），相关操作可访问阿里云官方地址：[https://www.aliyun.com/](https://www.aliyun.com/)（注册/登录）、[https://myaccount.console.aliyun.com/cert-info](https://myaccount.console.aliyun.com/cert-info)（实名认证页面）。

2. 使用主账号，或具备“管理员”“API-Key”权限的子账号登录（子账号无权限将无法创建API Key）。

## 二、开通阿里百炼服务

1. 访问阿里百炼官方控制台：https://dashscope.console.aliyun.com/ ，选择推荐地域（北京/新加坡优先）。

2. 点击“免费体验”，阅读并同意服务协议，确认开通服务。

3. 开通后可领取新用户免费Tokens额度（多模型合计最高约5000万，有效期180天），无需额外付费即可开始使用。

## 三、创建并获取API Key（核心步骤）

1. 从控制台进入“密钥管理”页面，切换至“API-Key”页签。

2. 点击“创建API-KEY”，弹出配置窗口后，按提示填写：


    - 归属账号：选择主账号或有权限的子账号。

    - 归属业务空间：无特殊需求（如分账、权限管控）选择默认空间即可。

    - 描述：填写备注（如“项目测试”“生产调用”），便于后续管理。

3. 点击“确定”，系统生成API Key（格式类似sk-xxxxxx），**该密钥仅展示一次**，立即复制并妥善保存（建议存至本地记事本，切勿泄露）。

## 四、后续管理与使用说明

1. API Key永久有效，删除后立即失效，可在“密钥管理”页面查看、禁用、删除已创建的密钥。

2. 单个业务空间最多可创建20个API Key，不同项目可创建不同密钥，便于区分用量。

3. 使用建议：调用时优先将API Key配置到环境变量，避免硬编码在代码中，降低泄露风险；新加坡地域调用需替换对应Base URL。

4. 权限说明：主账号可查看所有API Key，子账号仅能查看自身创建的密钥。

注意：API Key为个人/企业重要鉴权凭证，请勿公开或泄露，避免因未经授权使用产生安全风险或资金损失。

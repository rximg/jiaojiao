# 智谱AI（Zhipu）API Key申请简洁步骤（文档版）

## 一、前置准备

1. 注册并登录智谱AI开放平台，完成实名认证（个人/企业均可，为获取API Key的必要条件），相关操作可访问官方地址：[https://open.bigmodel.cn/](https://open.bigmodel.cn/)（注册/登录），实名认证在登录后个人中心完成，个人需提供真实姓名、身份证件信息，企业需提供企业信息及法人相关资料。

2. 确保登录账号为本人实名账号，无需额外权限配置，个人/企业实名账号均可直接创建API Key。

## 二、进入控制台（核心前置操作）

1. 访问智谱AI开放平台官网（[https://open.bigmodel.cn/](https://open.bigmodel.cn/)），使用已注册并完成实名认证的账号登录。

2. 登录成功后，点击页面右上角「控制台」，进入智谱AI开放平台控制台页面（控制台官网直达：[https://open.bigmodel.cn/overview](https://open.bigmodel.cn/overview)）。

## 三、创建并获取API Key（核心步骤）

1. 在控制台左侧菜单，找到「API密钥管理」（或点击右上角头像进入个人中心，在左侧菜单选择“项目管理”→“API Keys”），进入密钥管理页面。

2. 点击页面中的「新建密钥」（或“添加新的API Key”），弹出配置窗口后，按提示填写：

3. 密钥名称：填写备注信息（如“项目测试”“生产调用”），便于后续区分和管理。

4. 其他配置：无额外必填项，默认配置即可，无需修改额外参数。

5. 点击「确定」，系统立即生成API Key（格式类似sk-xxxxxxxxxxxxxxxx），**该密钥仅展示一次**，请立即复制并妥善保存（建议存至本地记事本，切勿泄露），用于后续API调用。

## 四、后续管理与使用说明

1. API Key永久有效，删除后立即失效，可在「API密钥管理」页面查看、禁用、激活、删除已创建的密钥，也可修改密钥名称和描述信息（密钥本身无法修改，需修改需删除后重新创建）。

2. 单个账号可创建多个API Key，无明确数量限制，不同项目可创建不同密钥，便于区分用量和管理，降低泄露风险。

3. 使用建议：调用时优先将API Key配置到环境变量，避免硬编码在代码中；通用API请求端点为[https://open.bigmodel.cn/api/paas/v4/](https://open.bigmodel.cn/api/paas/v4/)，使用GLM编码套餐需切换至专属Coding端点，调用时需在请求头配置对应鉴权信息。

4. 安全设置：可在密钥管理页面为API Key设置IP白名单，仅允许白名单内的IP地址调用该密钥，进一步提升安全性，白名单支持IP地址、IP段、IP地址范围三种格式。

注意：API Key为个人/企业重要鉴权凭证，请勿公开或泄露，避免因未经授权使用产生安全风险或资金损失；若发现密钥泄露，需立即删除该密钥并重新创建，防止损失扩大。
